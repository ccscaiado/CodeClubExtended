Introduction

In this project you will learn how to write a Python program. We will see:

  • the print function - useful for troubleshooting and user interaction
  • how Python handles text, special characters (e.g. quotation marks), and text that span multiple lines
  • the input function - a function that prompts a user for input
  • the concept of a variable and how it can be used to hold information
  • the int method - used to convert a number or string into a whole number
  • how to perform a few basic operations on numbers and strings/text

We also want you to get familiar with the idea of handling errors and troubleshooting. No one writes perfect code. You should expect quite a few error messages, there are mistakes intentionally inserted in this worksheet.

We will use Trinket, a web-based environment for coding.

Why Trinket? It is simple to use, it doesn't require any software installation, you can easily save and share your projects. Later on we will want to look at alternatives and that will depend on which direction you wish to push your projects.

Compatibility: It works in almost any browser including IE and Edge, and mobile browsers (Android and Apple).

Do I need a Trinket account? No. You can choose to create a free account (no need for premium) to save your projects or you can download files/save a link for later.

Do I have to use Trinket? No. There are other websites with similar environments or you can use the local installation in the CIS computers. If using the local installation, look for a Python 3 installation in the AppHub (ask a mentor for help).
